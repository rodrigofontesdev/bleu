# Box

Box é um tipo de componente que permite criar um efeito de "tom sobre tom" na interface.

Possui um fundo translúcido, com cantos arredondados e uma sombra suave ao seu redor.

Sendo capaz de se adaptar a cor do background, criando um leve contraste.

## Implementação no design

Box deve possuir um contraste suave em relação ao plano de fundo, a recomendação é aplicar uma
opacidade de 15% com a cor de background `#000000` e uma leve sombra externa. Deve ser suficiente
para criar o efeito esperado.

| Propriedade      | Valor                                      |
| :--------------- | :----------------------------------------- |
| Background color | `black.alpha.15`                           |
| Corner radius    | `8px`                                      |
| Outer shadow     | `X:0 Y:4px B:4px S:0 C:rgba(0, 0, 0, 25%)` |

## Quando usar?

O componente pode ser empregado em contextos onde um agrupamento de elementos precisa receber
destaque do restante dos componentes que formam a interface da aplicação.

<Primary />

:heavy_check_mark: **Uso correto**

- Destacar funcionalidade principal na interface
- Evidenciar contexto de um agrupamento de elementos

:x: **Uso incorreto**

- Utilizar em plano de fundo claro
- Manter baixo nível de contraste dos elementos internos

## Como usar?

O componente por padrão não renderiza uma tag semântica, opcionalmente é possível definir uma tag
atráves da propriedade `as`. É suportado qualquer elemento como descendente do Box.

### React

Certifique de possuir o pacote **@bleukit/react** instalado no projeto React. Para saber como
instalar confira o [guia de instalação](#).

```js
import { Box } from '@bleukit/react'
```
